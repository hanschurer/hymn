# Java literal

常见的 Java 字面量类型包括：

1. 整数类型：byte，short，int，long
2. 浮点数类型：float，double
3. 字符类型：char
4. 布尔类型：boolean

一些特殊的Literal
1. 制表符 \t   补齐空格
2. 空类型 null 转换成字符串打印出

定义变量的时候，如果加上final修饰符，这个变量就变成了常量
