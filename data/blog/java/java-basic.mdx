# Java literal

常见的 Java 字面量类型包括：
1. 整数类型字面量（Integer Literals）：如 10、-5 等。
2. 浮点数类型字面量（Floating-Point Literals）：例如 3.14、-0.5 。
3. 字符类型字面量（Character Literals）：像 'A'、'9' 。
4. 字符串类型字面量（String Literals）：比如 "Hello World" 。
5. 布尔类型字面量（Boolean Literals）：即 true 和 false 。
